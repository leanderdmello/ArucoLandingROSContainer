# Frame convention example

This example demonstrates the basic usage of the frame convention conversion functionality. The example code is written in C++ and uses the `creos_client` library. View the [main.cpp](src/main.cpp) file to see how to use the frame convention conversion functionality. The example specifically shows the usage of a convention conversion from ENU/FLU to NED/FRD of an odometry message.

## Usage

```bash
./build/bin/frame_convention_example
```
